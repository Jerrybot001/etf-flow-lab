# etf-flow-lab

ETF Flow Lab daily market flow website.

## 固定原則

- 固定模板，不因每日更新任意更換整體版型。
- 每日只更新資料、狀態、數字、文字與必要的小區塊。
- 未取得正式來源前，一律標示「待更新」或「待接正式來源」，不可用舊值或 Demo 假裝今日資料。

## 00981A 資料規則

- 00981A 第一順位來源為統一投信官網每日持股揭露。
- 因官網通常只揭露每日或最新資料，網站需自行保存每日官方快照。
- 歷史快照儲存在 `data/00981A/holdings-history.json`。
- 每日持股變化必須用「今日官方快照」減「前一個已保存官方快照」。
- 不使用 MoneyDJ、Yahoo、WantGoo、舊截圖或 Demo 資料作為今日 00981A 官方持股。

## 參考網站邊界

可參考 ETF Cross 的資訊組織方向，但不得抄襲版面、文字、互動流程或視覺設計。

可參考的概念：

- ETF 交集 / 重疊持股的分析入口。
- 主被動 ETF 異動歷史的資料切法。
- 產業權重輪動的呈現邏輯。
- 使用投信官網公開揭露資料，並清楚標示「僅供參考，請以官方公告為準」。

ETF Flow Lab 應維持自己的版型與語氣：盤前/盤後市場脈絡、三大法人、外資期貨、00981A 官方快照與每日變化。
